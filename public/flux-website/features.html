Features

What's inside

Flux ships with everything you need. No third-party packages required for the basics.

Interpreted by default
Run scripts instantly with zero compilation. Like Python, but with static types.
Native compilation
Compile to machine code via LLVM when you need performance. Same code, two modes.
System builtins
CPU info, memory stats, disk usage, process count â€” built into the language, not a library.
Structs + field access
Define data structures and access fields with dot notation. Clean and predictable.
String operations
String literals, concatenation with +, automatic type conversion with str().
Scoped variables
Lexical scoping with proper shadowing. Variables are resolved innermost-first.
HTTP server
Native POSIX HTTP server compiled directly to LLVM IR. No dependencies.
Pattern matching
Match expressions with exhaustiveness checking. Enums and algebraic types.
LSP support
Language server protocol for editor integration. Autocomplete and diagnostics.
