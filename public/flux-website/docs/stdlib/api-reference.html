Flux Standard Library — API Reference

Version: 0.1.0-draft

This document provides a comprehensive overview of every module in the Flux standard library. Each module lists its purpose, key types, and primary functions.

Core Modules

flux.io

Input/output operations for console, streams, and buffered I/O.

Item Signature Description
print(args...) fn print(args: any...) Print to stdout with newline
eprint(args...) fn eprint(args: any...) Print to stderr with newline
read_line(prompt?) fn read_line(prompt: String = "") -> String Read line from stdin
Stdin struct Stdin Standard input stream
Stdout struct Stdout Standard output stream
Stderr struct Stderr Standard error stream
BufReader<R> struct BufReader<R: Read> Buffered reader wrapper
BufWriter<W> struct BufWriter<W: Write> Buffered writer wrapper

Traits:

Trait Methods Description
Read read(buf: &mut [Byte]) -> Result<Int, IoError> Read bytes
Write write(buf: &[Byte]) -> Result<Int, IoError> Write bytes
Seek seek(pos: SeekPos) -> Result<Int, IoError> Seek to position
BufRead read_line() -> Result<String, IoError> Read a line

flux.http

HTTP client and server.

Type Description
Client HTTP client
Server HTTP server
Request HTTP request
Response HTTP response
Status HTTP status codes
Headers Header map 
// Client
let response = await http.get("https://api.example.com/data")
let data: MyType = response.json()?

// POST
let response = await http.post("https://api.example.com/data")
    .header("Content-Type", "application/json")
    .json({"key": "value"})
    .send()?
