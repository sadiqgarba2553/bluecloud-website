Getting Started with Flux

Welcome to Flux! This guide will get you from zero to your first running program in under five minutes.

1. Installation

Linux

curl -fsSL https://flux-lang.org/install.sh | sh

This installs flux to ~/.flux/bin and adds it to your PATH.

macOS

# Homebrew
brew install flux-lang

# Or the installer script
curl -fsSL https://flux-lang.org/install.sh | sh

Windows

# PowerShell
irm https://flux-lang.org/install.ps1 | iex

# Or download the MSI installer from https://flux-lang.org/download

From Source

git clone https://github.com/flux-bybluecloud/flux.git
cd flux
cargo build --release
# Add target/release to your PATH

Verify Installation

flux --version
# flux 0.1.0 (2026-06-29)

2. Your First Program

Create a Project

flux new hello_flux
cd hello_flux

This creates:

hello_flux/
├── Flux.toml
├── src/
│   └── main.flux
└── tests/
    └── main_test.flux

The Main File

Open src/main.flux:

fn main() {
    print("Hello, Flux! ⚡")
}

Run It

flux run

Output:

Hello, Flux! ⚡

3. Project Structure

File/Directory Purpose
Flux.toml Project manifest — name, version, dependencies
src/ Source code
src/main.flux Entry point for executables
src/lib.flux Entry point for libraries
tests/ Test files
examples/ Example programs
Flux.lock Locked dependency versions (auto-generated)

Flux.toml

[package]
name = "hello_flux"
version = "0.1.0"
edition = "2026"

[dependencies]
# Add dependencies here

4. Language Basics

Variables

let name = "Flux"          // Immutable (default)
mut counter = 0            // Mutable
const MAX = 100            // Compile-time constant

counter += 1               // OK — counter is mutable
// name = "Other"          // Error — name is immutable

Types

let age: Int = 30              // Integer
let pi: Float64 = 3.14159     // Float
let active: Bool = true        // Boolean
let letter: Char = 'A'        // Character
let greeting: String = "Hi"   // String

Functions

fn add(a: Int, b: Int) -> Int {
    return a + b
}

fn greet(name: String) {
    print("Hello, {name}!")
}

fn main() {
    let sum = add(3, 4)
    print("3 + 4 = {sum}")
    greet("World")
}

Control Flow

// If/else
if temperature > 30 {
    print("Hot!")
} else if temperature > 20 {
    print("Nice!")
} else {
    print("Cold!")
}

// For loop
for i in 0..5 {
    print(i)    // 0, 1, 2, 3, 4
}

for item in ["apple", "banana", "cherry"] {
    print(item)
}

// While loop
mut count = 0
while count < 10 {
    count += 1
}

// Match
match day {
    "Monday"    -> print("Start of week")
    "Friday"    -> print("Almost weekend!")
    "Saturday" | "Sunday" -> print("Weekend!")
    _           -> print("Midweek")
}

Structs

struct Person {
    name: String
    age: Int
}

impl Person {
    fn greet(self) {
        print("Hi, I'm {self.name}, age {self.age}")
    }
}

fn main() {
    let alice = Person { name: "Alice", age: 30 }
    alice.greet()
}

Error Handling

fn divide(a: Int, b: Int) -> Result<Int, String> {
    if b == 0 {
        return Err("Division by zero")
    }
    return Ok(a / b)
}

fn main() {
    match divide(10, 3) {
        Ok(result)  -> print("Result: {result}")
        Err(message) -> print("Error: {message}")
    }
}

5. IDE Setup

Visual Studio Code

  1. Install the Flux extension from the VS Code marketplace
  2. Open a Flux project folder
  3. You'll get:
  4. Syntax highlighting
  5. Autocomplete
  6. Error diagnostics
  7. Go-to-definition
  8. Hover documentation
  9. Auto-formatting on save

JetBrains IDEs

  1. Go to Settings → Plugins → Marketplace
  2. Search for Flux Language
  3. Install and restart

6. Essential Commands

Command Description
flux new <name> Create a new project
flux build Compile the project
flux run Build and run
flux test Run tests
flux format Format all source files
flux lint Run the linter
flux add <package> Add a dependency
flux docs Generate documentation
flux clean Remove build artifacts
