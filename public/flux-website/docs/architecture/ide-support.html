Flux IDE Support

1. Overview

Flux provides first-class IDE support through the Language Server Protocol (LSP) and Debug Adapter Protocol (DAP). Official extensions are provided for VS Code and JetBrains IDEs.

2. Language Server (flux-lsp)

2.1 Architecture

IDE (VS Code / JetBrains / Neovim / etc.)
    │
    │  LSP Protocol (JSON-RPC over stdio/TCP)
    ▼
flux-lsp (Language Server)
    │
    ├── Lexer + Parser (incremental)
    ├── Semantic Analysis (incremental)
    ├── Type Checker (incremental)
    └── Query Engine (cached)

2.2 Supported LSP Features

2.3 Completion Capabilities

The completion engine provides:

  • Keyword completion: Context-appropriate keywords
  • Type completion: Available types from imports and current scope
  • Method completion: Methods available on the current type
  • Field completion: Fields of structs/classes
  • Import completion: Available modules and packages
  • Snippet completion: Common code patterns
  • Postfix completion: .if, .match, .let on expressions
user.    →  .name, .email, .age, .greet(), .is_adult()
items.fi →  .filter(), .find(), .first()
Result.  →  .Ok(), .Err(), .is_ok(), .is_err()

4. VS Code Extension

4.1 Extension ID

flux-lang.flux

4.2 Features

  • Syntax highlighting (TextMate grammar + semantic tokens)
  • Full LSP integration
  • Integrated debugger (DAP)
  • Test explorer integration
  • Task provider (flux build, flux test, flux run)
  • Problem matcher for compiler output
  • Snippets for common patterns
  • Status bar (compiler version, target, errors/warnings count)

4.3 TextMate Grammar Scopes

keyword.declaration.flux fn, let, mut, const, struct, class, enum, trait

5. JetBrains Plugin

5.1 Plugin ID

com.flux-lang.flux

5.2 Features

  • PSI-based parser for syntax analysis
  • Full LSP integration as backend
  • Run configurations (build, run, test)
  • Integrated debugger
  • Code structure view
  • Refactoring support
  • Inspections and quick-fixes
  • Flux REPL tool window
  • New Flux project wizard

5.3 Supported IDEs

  • IntelliJ IDEA (Community and Ultimate)
  • CLion
  • WebStorm
  • All JetBrains IDEs (via plugin)
